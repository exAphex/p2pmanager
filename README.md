# P2PManager

P2PManager provides an overview of your p2p and crypto investments at a central place.

## Supported Platforms

### P2P

- GetIncome
- Bondster
- Esketit
- EstateGuru
- LendSecured
- Lendermarket
- PeerBerry

### Cryptos

- ATOM
- Cronos
- KAVA
- Osmosis
- Solana
- Terra LUNA
